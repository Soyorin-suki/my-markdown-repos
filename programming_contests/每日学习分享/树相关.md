# 树相关知识
## 树的存储
首先，树也是一种特殊的图，所以图的储存方式是完全适用的，不再赘述
1. 可以只记录父节点：
    只用一个parent[N]数组记录。这种方式可以获得的信息较少，而且不便于自上而下的遍历。常用于自底向上的递推问题中。
2. 左孩子右兄弟
   每层记录第一个节点，每个节点记录与它相同的下一个兄弟的位置
3. **最常用于二叉树的**定义一个数组包括left与right

## 树的四种遍历
1. 先序遍历
   简单来说就是递归（或栈）中先输出这个节点在进入左子结点，再进入右子节点直到叶子。
   ``` 
   void preorder(ll x){
    if(x==0)return;
    cout<<tree[x].val;
    preorder(tree[x].ls);
    preorder(tree[x].rs);
   }
   ```
2. 中序遍历
   同上，只不过就是先进入左子节点再输出该点，最后进入右子节点，到达叶子返回
3. 后序遍历
   同上，但是先进入左子节点再进入右子节点，最后再输出该点
4. 层序遍历
   就是BFS

可以从中序遍历和先序遍历（或中序遍历和后序遍历）中推出树的形状
