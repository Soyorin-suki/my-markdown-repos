# 康托展开
**康托展开**用于求一个排列在所有1~n的排列之间的字典序排名
设有排列$p=a_1a_2\dots a_n$,那么对于任意字典序排名比p小的排列，一定存在$i$,使得前$i-1(1\le i\le n)$位与$p$对应位相同，第$i$位比$p_i$小，后续位随意。
所以对于任意i，满足条件的排列数就是$A_i\cdot(n-i)!$（$A_i$表示$a_i$后面比$a_i$小的数字的数量）
遍历$i$即可得到方案数$\sum_{i=1}^{n-1}A_i\cdot (n-i)!$,再加上1即是排名

朴素实现$O(n^2)$
可用树状数组二维偏序加速至$O(n\log n)$
