# 关于格点计数
> 以下所提到的格点计数中如非特殊说明，每步均只能向上或向右。
> 以下如非特殊说明均为从$(0,0)$为起始点的，其他情况可酌情平移得到。


## 1.一般的格点计数
对于最一般的从$(0,0)\rightarrow (a,b)$的路径的计数，可以通过在$a+b$中操作中选出$a$个操作向右得到。
即：
$$S_1(a,b)=\binom{a+b}{a}$$
