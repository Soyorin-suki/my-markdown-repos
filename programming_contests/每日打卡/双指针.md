### 例题1：[字符串](https://ac.nowcoder.com/acm/problem/18386)
```题目描述
小N现在有一个字符串S。他把这这个字符串的所有子串都挑了出来。
一个S的子串T是合法的，当且仅当T中包含了所有的小写字母。
小N希望知道所有的合法的S的子串中，长度最短是多少。
```
```输入描述
输入描述：
一行一个字符串S。只包含小写字母。S的长度不超过106.
```
```输出描述
输出描述：
一行一个数字，代表最短长度。数据保证存在一个合法的S的子串。
```
```实例
输入：ykjygvedtysvyymzfizzwkjamefxjnrnphqwnfhrnbhwjhqcgqnplodeestu
输出：49
```
实现代码：
```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;
int c[26];
inline bool check(void){
    int i=0;
    for(i=0;i<26;++i){
        if(!c[i])return false;
    }
    return true;
}
int main(void){
    string s;
    cin>>s;
    int i,left=0,right=0,m=s.length(),ans=m;
    char temp;
    while(right>=left&&right<=m){
        if(!check()){
            c[s[right]-'a']++;
            ++right;
        }
        else {
            c[s[left]-'a']--;
            ans=min(ans,right-left);
            ++left;
        }
        
    }
    cout<<ans;
    
    return 0;
}
```

### 例题2：[丢手绢](https://ac.nowcoder.com/acm/problem/207040)
```
“丢~丢~丢手绢，轻轻地放在小朋友的后面，大家不要告诉她，快点快点抓住她，快点快点抓住她。”
牛客幼儿园的小朋友们围成了一个圆圈准备玩丢手绢的游戏，但是小朋友们太小了，不能围成一个均匀的圆圈，即每个小朋友的间隔可能会不一致。为了大家能够愉快的玩耍，我们需要知道离得最远的两个小朋友离得有多远（如果太远的话牛老师就要来帮忙调整队形啦！）。
因为是玩丢手绢，所以小朋友只能沿着圆圈外围跑，所以我们定义两个小朋友的距离为沿着圆圈顺时针走或者逆时针走的最近距离。
```
```输入描述
输入描述：
第一行一个整数N，表示有N个小朋友玩丢手绢的游戏。
接下来的第2到第n行，第i行有一个整数，表示第i-1个小朋友顺时针到第i个小朋友的距离。
最后一行是第N个小朋友顺时针到第一个小朋友的距离。
```
```输出描述
输出描述：
输出一个整数，为离得最远的两个小朋友的距离。
```
```实例
输入：
3
1
2
3
输出：3
```
实现代码：
```cpp
#include <iostream>
#include <algorithm>
using namespace std;
typedef long long ll;
const ll N=1e5+10;
ll a[N];
int main(void){
    ll n,i,ans=0,temp=0,sum=0;
    cin>>n;
    for(i=0;i<n;++i){
        cin>>a[i];
        sum+=a[i];
    }
    ll left=0,right=0;
    while(left<=n){
        while((temp*2)<=sum){
            ans=max(temp,ans);
            temp+=a[(right)%n];
            ++right;
        }
        temp-=a[left];
        ++left;
    }
    cout<<ans;
    
    
    return 0;
}
```
